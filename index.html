Integration

You can use the Elusive icons font with bootstrap or without it.

Bootstrap using LESS

Use this method if integrating with Twitter Bootstrap using LESS

  1. Copy the elusive-iconfont directory into your project.
  2. Copy elusive-webfont.less into your bootstrap/less directory.
  3. Open bootstrap.less and replace @import "sprites.less"; with @import "elusive-webfont.less";
  4. Open your project's elusive-webfont.less and edit the font url to ensure it points to the right place. 
    @font-face {
  font-family: 'Elusive-Icons';
  src:url('../font/Elusive-Icons.eot');
  src:url('../font/Elusive-Icons.eot?#iefix') format('embedded-opentype'),
    url('../font/Elusive-Icons.svg#Elusive-Icons') format('svg'),
    url('../font/Elusive-Icons.woff') format('woff'),
    url('../font/Elusive-Icons.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}
  5. Re-compile your LESS if using a static compiler. Otherwise, you should be good to go.
  6. Check out the examples to start using Elusive Icons!

Bootstrap using CSS

Use this method if integrating with Twitter Bootstrap using CSS.

  1. Customize Twitter Bootstrap here. Make sure to uncheck the default "Icons" under "Base CSS."
  2. Copy the Elusive Icons font directory into your project.
  3. Copy elusive-bootstrap.css into your project.
  4. Open your project's elusive-bootstrap.css and edit the font url to ensure it points to the right place (see above example).
  5. In the <head> of your html, reference the location to your elusive-bootstrap.css. 
    <link rel="stylesheet" href="../css/bootstrap.css">
<link rel="stylesheet" href="../css/elusive-bootstrap.css">
  6. Check out the examples to start using Elusive Icons!

Not using Bootstrap?

Elusive Icons works just as well without Twitter Bootstrap.

  1. Copy the Elusive Icons directory into your project.
  2. Copy elusive-webfont.less or elusive-bootstrap.css into your project.
  3. Open your project's elusive-webfont.less or elusive-bootstrap.css and edit the font url to ensure it points to the right place (see above example).
  4. Check out the examples to start using Elusive Icons!

Code

Inline Icon

Place Elusive Icons just about anywhere with the <i> tag.

icon-globe 
<div style="font-size: 24px;">
  <i class="icon-globe"></i> icon-globe
</div>
Icon classes are echoed via CSS :before.

Increase the icon size by using the icon-large class. This increases the size by 33% relative to the font-size of the container.

icon-globe 
<div style="font-size: 24px;">
  <i class="icon-globe icon-large"></i> icon-globe
</div>
